## 📌 Project Overview
This project consists of a suite of five Bash scripts designed to perform an audit of a Linux system with a focus on Open Source Software (FOSS) principles and specifically the **Git** version control system. 

The scripts range from basic system identification to log analysis and the generation of a personal open-source manifesto.

---

## 📂 Script Descriptions

| Script Name | Purpose | Key Features |
| :--- | :--- | :--- |
| **Script_1.sh** | System Identity Report | Displays Kernel version, Distro info, and Uptime. |
| **Script_2.sh** | FOSS Package Inspector | Checks if Git is installed using `dpkg` or `rpm`. |
| **Script_3.sh** | Directory & Permission Auditor | Audits permissions and sizes of system dirs and Git configs. |
| **Script_4.sh** | Log File Analyzer | Counts keyword occurrences (like "error") in any log file. |
| **Script_5.sh** | Manifesto Generator | Creates a personalized `.txt` file based on user input. |

---

## 🛠 Prerequisites & Dependencies
To run these scripts, ensure you have:
* A **Linux-based OS** (Ubuntu, Debian, Fedora, etc.)
* **Bash Shell** (Standard on most Linux distros)
* **Git** (For Script 2 and 3 to show full results)
* **Permissions:** Some scripts may require `sudo` to read specific system directories (like `/var/log`).

---

## 🚀 Step-by-Step Instructions

### 1. Preparation
First, navigate to your script directory and ensure all files have execution permissions:
```bash
cd Script
chmod +x Script_*.sh
```

### 2. Running the Scripts

#### **Script 1: System Identity**
Generates a report of who you are and what system you are running.
```bash
./Script_1.sh
```

#### **Script 2: Package Inspector**
Checks if the chosen software (Git) is installed and shows its metadata.
```bash
./Script_2.sh
```

#### **Script 3: Directory Audit**
Checks permissions for core Linux folders and looks for your Git configuration files.
```bash
./Script_3.sh
```

#### **Script 4: Log Analyzer**
Analyzes a log file. You must provide a file path as an argument.
* *Usage:* `./Script_4.sh <file_path> <keyword>`
```bash
# Example: Search for "error" in the syslog
./Script_4.sh /var/log/syslog error
```

#### **Script 5: Manifesto Generator**
An interactive script that asks for your input to generate a text file.
```bash
./Script_5.sh
```
